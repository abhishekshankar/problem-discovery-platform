# Pre-launch legal review (PRD §17.3)

Schedule **before** relying heavily on Tier-3 scraped sources or high-volume public indexing.

**Suggested checklist**

- ToS posture for each Tier-3 source (G2, Capterra, etc.)
- GDPR applicability if storing EU-origin attributable data
- Copyright / fair use for stored verbatim quotes in briefs
- Recent scraping / data-access case law (e.g. Meta v. Bright Data; NYT v. OpenAI — discuss with counsel, not legal advice)

**Budget:** plan for roughly $500–1000 for a one-hour specialist consult as noted in the PRD.
